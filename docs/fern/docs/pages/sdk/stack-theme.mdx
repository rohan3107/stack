# StackTheme

Applies a custom theme to the Stack components.

## Props

- `theme`: An object containing theme configurations for light and dark modes, and a radius property.
  - `light`: Object containing color configurations for light mode.
  - `dark`: Object containing color configurations for dark mode.
  - `radius`: Defines the border radius for components.
- `children`: `React.ReactNode` - The child components to which the theme will be applied.
- `nonce`: `string` - A unique identifier used for security purposes in the generated style tag.

## Example

```tsx
import { StackTheme } from '@stackframe/stack';

const CustomTheme = () => (
  <StackTheme
    theme={{
      light: {
        background: '#ffffff',
        foreground: '#000000',
        // ... other color properties
      },
      dark: {
        background: '#000000',
        foreground: '#ffffff',
        // ... other color properties
      },
      radius: '0.5rem'
    }}
    nonce="unique-nonce-value"
  >
    {/* Your app components */}
  </StackTheme>
);
```
