# MagicLinkSignIn

A component that handles magic link sign-in functionality.

## Props

This component does not accept any props.

## Example

```tsx
import { MagicLinkSignIn } from '@stackframe/stack';

function SignInPage() {
  return (
    <div>
      <h1>Sign In</h1>
      <MagicLinkSignIn />
    </div>
  );
}
```

This component renders a form for email input and handles the magic link sign-in process. It also includes OTP verification if needed.
