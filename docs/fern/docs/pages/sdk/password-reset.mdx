# PasswordReset

A component for handling password reset functionality, displaying appropriate messages based on the reset link status.

## Props

- `searchParams`: `Record<string, string>` - An object containing search parameters, including the reset code.
- `fullPage` (optional): `boolean` - Determines if the component should be displayed as a full page. Default is `false`.

## Example

```tsx
import { PasswordReset } from '@stackframe/stack';

function ResetPasswordPage() {
  const searchParams = { code: 'reset_code_here' };

  return (
    <PasswordReset
      searchParams={searchParams}
      fullPage={true}
    />
  );
}
```
