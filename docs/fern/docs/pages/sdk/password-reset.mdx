# PasswordReset

A component for handling password reset functionality. It validates the reset code and displays appropriate messages or the reset form.

## Props

- `searchParams`: `Record<string, string>` - An object containing search parameters, including the reset code.
- `fullPage` (optional): `boolean` - Determines if the component should be displayed in full-page mode. Default is `false`.

## Example

```tsx
import { PasswordReset } from '@stackframe/stack';

function PasswordResetPage() {
  const searchParams = {
    code: 'reset_code_here',
  };

  return (
    <PasswordReset
      searchParams={searchParams}
      fullPage={true}
    />
  );
}
```
