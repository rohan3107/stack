# SelectedTeamSwitcher

A component that renders a dropdown for switching between teams.

## Props

- `urlMap` (optional): `(team: Team) => string` - A function that maps a team to a URL.
- `selectedTeam` (optional): `Team` - The currently selected team.
- `noUpdateSelectedTeam` (optional): `boolean` - If true, prevents updating the selected team in the user's preferences.

## Example

```tsx
import { SelectedTeamSwitcher } from '@stackframe/stack';

function TeamSelector() {
  const handleUrlMap = (team) => `/dashboard/${team.id}`;

      noUpdateSelectedTeam={false}
```
