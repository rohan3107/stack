# EmailVerification

A component for handling email verification processes, displaying appropriate messages based on the verification status.

## Props

- `searchParams` (optional): `Record<string, string>` - URL search parameters, expected to contain a `code` for verification.
- `fullPage` (optional): `boolean` - Determines if the component should be displayed in full-page mode.

## Example

```tsx
import { EmailVerification } from '@stackframe/stack';

function VerificationPage() {
  const searchParams = { code: 'verification_code_here' };

  return (
    <EmailVerification
      searchParams={searchParams}
      fullPage={true}
    />
  );
}
```
