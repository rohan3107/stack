# StackHandler

Handles routing and rendering of authentication-related components based on the provided stack path.

## Props

- `app`: `StackServerApp<HasTokenStore>` - The Stack server application instance.
- `params`: `{ stack?: string[] }` - The stack path parameters.
- `searchParams`: `Record<string, string>` - URL search parameters.
- `fullPage`: `boolean` - Whether to render the component in full-page mode.
- `componentProps`: `{ [K in keyof Components]?: Parameters<Components[K]> }` - Props for individual components.

## Example

```tsx
import { StackHandler } from '@stackframe/stack';

export default function Page({ params, searchParams }) {
  return (
    <StackHandler
      app={myStackServerApp}
      params={params}
      searchParams={searchParams}
      fullPage={true}
      componentProps={{
        SignIn: { /* SignIn component props */ },
        SignUp: { /* SignUp component props */ },
        // Add props for other components as needed
      }}
    />
  );
}
```
