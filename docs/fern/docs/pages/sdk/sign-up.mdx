# SignUp

The `SignUp` component is a pre-built component that displays all the sign-up methods available based on the Stack dashboard configuration.

## Props

- `fullPage` (optional): `boolean` - Determines whether the component should occupy the full page. Default is `false`.
- `automaticRedirect` (optional): `boolean` - Enables automatic redirection for signed-in users.
- `noPasswordRepeat` (optional): `boolean` - Removes the password repeat field from the sign-up form.
- `extraInfo` (optional): `React.ReactNode` - Additional information to display in the sign-up form.
- `firstTab` (optional): `'magic-link' | 'password'` - Sets the initially active tab in the sign-up form.

<Note>
  When `automaticRedirect` is not used, you can use the `useUser` hook to check the user's sign-in status and perform manual redirection if necessary.
</Note>

## Example

```tsx
import { SignUp } from '@stackframe/stack';

function SignUpPage() {
  return (
    <SignUp
      fullPage={true}
      automaticRedirect={true}
      noPasswordRepeat={false}
      extraInfo={<p>Additional information here</p>}
      firstTab="magic-link"
    />
  );
}
```
