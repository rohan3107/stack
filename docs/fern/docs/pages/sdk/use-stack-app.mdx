# useStackApp

A React hook that retrieves the current Stack app instance from the StackAppContext.

## Example

```tsx
import { useStackApp } from '@stackframe/stack';

function MyComponent() {
  const app = useStackApp();
  
  // Now you can use the app instance
  // For example:
  // const user = app.auth.getUser();
}
```

Note: Make sure to use this hook within a component that is wrapped by a StackAppProvider.
