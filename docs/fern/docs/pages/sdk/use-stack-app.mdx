# useStackApp

A hook that provides access to the StackClientApp object and related functionality.

## Returns

- `isReady`: `boolean` - Indicates if the app is ready.
- `appInstance`: `StackClientApp | undefined` - The StackClientApp object.
- `selectChain`: `(chainName: string) => void` - Function to select a blockchain.
- `connectWallet`: `() => void` - Function to open the wallet connection dialog.

## Example

```tsx
import { useStackApp } from '@stackframe/stack';

function MyComponent() {
  const { isReady, appInstance, selectChain, connectWallet } = useStackApp();

  if (!isReady) return <div>Loading...</div>;

  return (
    <div>
      <div>Sign In URL: {appInstance?.urls.signIn}</div>
      <button onClick={() => selectChain('ethereum')}>Select Ethereum</button>
      <button onClick={connectWallet}>Connect Wallet</button>
    </div>
  );
}

The `useStackApp` hook returns the `StackClientApp` object that you can use to interact with the Stack API. If you want to learn more about the `StackClientApp` object, check out the [App](./stack-app.mdx) documentation.

Example:

```jsx
import { useStackApp } from "@stackframe/stack";

function MyComponent() {
  const stackApp = useStackApp();
  return <div>Sign In URL: {stackApp.urls.signIn}</div>;
}
```
