# ForgotPassword

A component that renders a forgot password form or related messages based on the user's state.

## Props

- `fullPage` (optional): `boolean` - Determines if the component should be displayed in full-page mode.

## Example

```tsx
import { ForgotPassword } from '@stackframe/stack';

// Basic usage
<ForgotPassword />

// Full-page mode
<ForgotPassword fullPage={true} />
```
