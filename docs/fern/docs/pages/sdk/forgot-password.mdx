# ForgotPassword

A component that renders a forgot password form with optional full-page layout.

## Props

- `fullPage` (optional): `boolean` - Determines if the component should be rendered in full-page mode.

## Example

```tsx
import { ForgotPassword } from '@stackframe/stack';

// Default usage
<ForgotPassword />

// Full-page mode
<ForgotPassword fullPage={true} />
```
