# SignIn

Displays a pre-built UI component for user sign-in with configurable options.

## Props

- `fullPage` (optional): `boolean` - Determines whether the component should occupy the full page. Default is `false`.
- `automaticRedirect` (optional): `boolean` - Enables automatic redirection for signed-in users when set to `true`.
- `extraInfo` (optional): `React.ReactNode` - Additional information to display in the sign-in component.
- `firstTab` (optional): `'magic-link' | 'password'` - Specifies the initial tab to display. Can be either 'magic-link' or 'password'.

## Example

```tsx
import { SignIn } from '@stackframe/stack';

function SignInPage() {
  return <SignIn fullPage automaticRedirect />;
}

The `SignIn` component is a pre-built UI element that displays all available sign-in methods as configured in the Stack dashboard.

<Note>
  This component does not redirect a signed-in user. To achieve automatic redirection for signed-in users, you can use the `useUser` hook to check the user's sign-in status and perform the redirection if necessary.
</Note>
